# Nexter - Advanced CSS Grid & SCSS Project

A modern, fully responsive real estate landing page built using **advanced CSS Grid techniques and SCSS architecture**.  
This project is part of Jonas Schmedtmann’s Advanced CSS & Sass course and demonstrates real-world frontend layout skills.

---

## 🚀 Project Overview

Nexter is a high-end real estate website UI featuring a complex grid-based layout system, modular SCSS architecture, and responsive design principles. The project focuses heavily on mastering **CSS Grid layout patterns used in production-level interfaces**.

---

## 🧠 Key Skills Demonstrated

### 🎯 CSS Grid Mastery
- Advanced grid layout using named grid lines
- Full-page layout structure using grid areas
- Nested grid systems for components (cards, gallery, sections)
- Responsive grid reflow using media queries
- Complex image gallery grid positioning
- Overlapping grid elements and layering techniques

---

### 🎨 SCSS (Sass) Architecture
- Modular SCSS file structure (7-1 pattern)
- Variables for colors, typography, and spacing
- Nesting for component-based styling
- Partial files for maintainability
- Reusable design system approach
- SCSS compilation using npm scripts

---

### 📱 Responsive Web Design
- Mobile-first responsive breakpoints
- Adaptive grid restructuring across screen sizes
- Fluid typography scaling using rem units
- Flexible layout transitions from desktop to mobile

---

### 🧩 Component-Based UI Design
- Reusable card components (homes section)
- Feature icon grid system
- Sidebar navigation layout
- Hero header with overlay and background layering
- Structured testimonial/story section

---

### ⚙️ Frontend Workflow & Tooling
- Node.js + npm project setup
- Sass compiler integration via npm scripts
- Live SCSS watch mode (`npm run start`)
- Production CSS build process
- Organized asset pipeline (images, SVG sprite usage)

---

### 🎭 Advanced CSS Techniques
- CSS variables and custom design tokens
- Flexbox for alignment inside grid items
- SVG icon system using `<use>` sprite references
- Background overlays with gradients + images
- Object-fit and responsive image handling
- Hover states and UI transitions
